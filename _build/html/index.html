WordPressって何?¶

「WordPress」とはブログやCMSを作るためのオープンソースのソフトウェアです。

WordPress (ワードプレス) は、オープンソースのブログ／CMS プラットフォームです。セマンティック Web、コードやデザインの美しさ、Web 標準、ユーザビリティなどを意識して開発されており、無料でダウンロードして使うことができます。
